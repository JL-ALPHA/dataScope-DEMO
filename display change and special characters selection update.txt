CHANGE #6: ENHANCED SPECIAL CHARACTER ANALYSIS
----------------------------------------------

Location: Line ~700-780 in datascope_UI.py (Special Character Analysis section in analysis_handler)

MAJOR ENHANCEMENT:
The Special Character Analysis was completely rewritten to provide detailed breakdown:

NEW FEATURES ADDED:
1. Separation of ASCII vs Non-ASCII special characters
2. Character type counting and frequency analysis
3. Sample character display with truncation
4. Improved data table display with multiple columns

NEW COLUMNS IN DATA TABLE:
- Column: Name of the analyzed column
- Total Special Chars: Total unique special characters found
- ASCII Special Count: Count of ASCII special characters (ord < 128)
- Non-ASCII Count: Count of non-ASCII special characters (ord >= 128)
- ASCII Characters: Sample of ASCII special characters found (first 15)
- Non-ASCII Characters: Sample of non-ASCII characters found (first 10)
- ASCII Frequency: Total occurrences of ASCII special characters in data
- Non-ASCII Frequency: Total occurrences of non-ASCII characters in data

CODE STRUCTURE:
```python
# Separate ASCII and Non-ASCII special characters
ascii_special = []
non_ascii_special = []

for char in set(all_text):
    if not char.isalnum() and not char.isspace():
        if ord(char) < 128:  # ASCII range
            ascii_special.append(char)
        else:  # Non-ASCII
            non_ascii_special.append(char)

# Count total special characters
total_special = len(ascii_special) + len(non_ascii_special)

# Get counts of each character type in the actual data
ascii_count = sum(all_text.count(char) for char in ascii_special)
non_ascii_count = sum(all_text.count(char) for char in non_ascii_special)
```
